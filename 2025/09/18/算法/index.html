1. 二分查找

两数之和

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        n = len(numbers)
        for i in range(n):
            low, high = i +1, n-1
            while low <= high:
                mid = (low + high) // 2
                if numbers[mid] == target - numbers[i]:
                    return [i+1, mid+1]
                elif numbers[mid] > target - numbers[i]:
                    high = mid - 1
                else:
                    low = mid + 1
        return [-1, -1]

二分查找只能用于有序数组，时间复杂度为O(logn)，对于两数之和问题，我们首先遍历数组找到第一个数O(n)，然后用二分查找在有序数组里找到第二个数O(logn)，总为O(nlogn)

2. 双指针

只有一个输入，从两端开始遍历

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        low, high = 0, len(numbers) - 1
        while low < high:
            total = numbers[low] + numbers[high]
            if total == target:
                return [low + 1, high + 1]
            elif total < target:
                low += 1
            else:
                high -= 1

        return [-1, -1]

时间复杂度为O(n)

class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        """
        Do not return anything, modify nums1 in-place instead.
        """
        sorted = []
        p1, p2 = 0, 0
        while p1 < m or p2 < n:
            if p1 == m:
                sorted.append(nums2[p2])
                p2 = p2 + 1
            elif p2 == n:
                sorted.append(nums1[p1])
                p1 = p1 + 1
            elif nums1[p1] < nums2[p2]:
                sorted.append(nums1[p1])
                p1 = p1 + 1
            else:
                sorted.append(nums2[p2])
                p2 = p2 + 1
        nums1[:] = sorted

时间复杂度O(m+n)

（1）子数组最大平均数

给你一个由 n 个元素组成的整数数组 nums 和一个整数 k 。

请你找出平均数最大且 长度为 k 的连续子数组，并输出该最大平均数。

任何误差小于 10-5 的答案都将被视为正确答案。

示例 1：

输入：nums = [1,12,-5,-6,50,3], k = 4
输出：12.75
解释：最大平均数 (12-5-6+50)/4 = 51/4 = 12.75

示例 2：

输入：nums = [5], k = 1
输出：5.00000

解答：

class Solution:
    def findMaxAverage(self, nums: List[int], k: int) -> float:
        start = 0
        sum_, max_avg = 0, -math.inf
        for end, x in enumerate(nums):
            sum_ = sum_ + x
            if end + 1 - start >= k:
                max_avg = max(max_avg, (sum_ / k))
                sum_ = sum_ - nums[start]
                start = start + 1
        return max_avg

（2）至多包含两个不同字符的最长子串

给你一个字符串 s ，请你找出 至多 包含 两个不同字符 的最长子串，并返回该子串的长度。

示例 1：

输入：s = "eceba"
输出：3
解释：满足题目要求的子串是 "ece" ，长度为 3 。

示例 2：

输入：s = "ccaabbb"
输出：5
解释：满足题目要求的子串是 "aabbb" ，长度为 5 。

循环的条件：不符合题意（字串里面有超过三个的不同字符）

解答：

最长类问题：

  • 先扩张窗口，让它尽可能大；
  • 如果超出限制（非法），再收缩。
  • while 条件 → 窗口不合法时收缩。
class Solution:
    def lengthOfLongestSubstringTwoDistinct(self, s: str) -> int:
        start = 0
        max_len, hashmap = 0, {}

        for end in range(len(s)):
            tail = s[end]
            hashmap[tail] = hashmap.get(tail, 0) + 1
            while len(hashmap) > 2:
                head = s[start]
                hashmap[head] -= 1
                if hashmap[head] == 0:
                    del hashmap[head]
                start += 1
            max_len = max(max_len, end + 1 - start)
        return max_len

（3）无重复字符的最长字串

给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

解答：

  • 最长类问题：
    • 先扩张窗口，让它尽可能大；
    • 如果超出限制（非法），再收缩。
    • while 条件 → 窗口不合法时收缩。
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        start = 0
        hashmap = {}
        max_len = 0
        for end in range(len(s)):
            hashmap[s[end]] = hashmap.get(s[end], 0) + 1
            while end + 1 - start > len(hashmap):
                head = s[start]
                hashmap[head] -= 1
                start += 1
                if hashmap[head] == 0:
                    del hashmap[head]
            max_len = max(max_len, end + 1 - start)

        return max_len

（4）和大于某数值的长度最小的子数组

给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其总和大于等于 target 的长度最小的 子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度**。**如果不存在符合条件的子数组，返回 0 。

示例 1：

输入：target = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。

示例 2：

输入：target = 4, nums = [1,4,4]
输出：1

示例 3：

输入：target = 11, nums = [1,1,1,1,1,1,1,1]
输出：0

解答：

最短类问题：

  • 先扩张窗口，直到满足条件；
  • 一旦满足条件，尝试收缩，保证尽可能小。
  • while 条件 → 窗口合法时收缩。
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        start  = 0
        sum_ = 0
        min_len = math.inf
        for end in range(len(nums)):
            sum_ += nums[end]
            while sum_ >= target:
                min_len = min(min_len, end + 1 -start)
                sum_ -= nums[start]
                start += 1
        if min_len == math.inf:
            return 0
        return min_len

（5）找到字符串中所有字母异位词

给定两个字符串 s 和 p，找到 s 中所有 p 的 异位词 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

示例 1:

输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。

示例 2:

输入: s = "abab", p = "ab"
输出: [0,1,2]
解释:
起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。

解答：

  • 窗口长度固定，判断条件用if
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        start = 0
        hashmap_, hashmap = {}, {}
        ans = []
        # 统计 p 中每个字符的频率
        for i, x in enumerate(p):
            hashmap_[x] = hashmap_.get(x, 0) + 1

        # 遍历 s，用滑动窗口检查子串是否是 p 的字母异位词
        for end in range(len(s)):
            # 扩展窗口：加入右边的字符
            hashmap[s[end]] = hashmap.get(s[end], 0) + 1

            # 如果窗口和目标频率表相等，说明找到一个异位词
            if hashmap == hashmap_:
                ans.append(start)

            # 保持窗口长度不超过 len(p)
            if len(p) <= end - start + 1:
                hashmap[s[start]] -= 1
                if hashmap[s[start]] == 0:
                    del hashmap[s[start]]
                start += 1

        return ans

最长类问题：超出限制，再收缩（while条件为不符合）

class Solution:
    def maximumUniqueSubarray(self, nums: List[int]) -> int:
        start = 0
        sum_ = 0
        ans = 0
        hashmap = {}
        for end in range(len(nums)):
            # 从这里开始是子串扩张
            sum_ += nums[end]
            hashmap[nums[end]] = hashmap.get(nums[end], 0) + 1
            if len(hashmap) == end + 1 - start:
                # 满足条件时更新答案
                ans = max(ans, sum_)
            while len(hashmap) < end + 1 - start:
                head = nums[start]
                # 子串缩小，记得删除头元素
                sum_ -= head
                hashmap[head] -= 1
                if hashmap[head] == 0:
                    del hashmap[head]
                start += 1
        return ans
